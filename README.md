[PHPoole](http://phpoole.narno.org) demo plugin.

Installation
------------

Add this to your PHPoole ```composer.json``` file:

    {
      "require": {
        "phpoole/plugin-demo": "master-dev"
      },
      "repositories": [
        {
          "type": "vcs",
          "url": "https://github.com/PHPoole/plugin-Demo"
        }
      ]
    }

Usage
-----

Run a new website and read the console.
